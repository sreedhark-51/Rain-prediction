⚠️ TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

Issue: "ModuleNotFoundError: No module named 'seaborn'"
Solution: Run `pip install -r requirements.txt`

Issue: "Model file not found"
Solution: The model is already trained, check models/flood_model.pkl exists
         If missing, run: python train_model.py

Issue: "Port 5000 already in use"
Solution: Either stop other Flask apps, or edit app.py line with:
         app.run(debug=False, host='0.0.0.0', port=5001)

Issue: "Connection refused" when accessing localhost:5000
Solution: Make sure Flask app is running (python app.py)
         Check console for error messages

Issue: Predictions show unusual results
Solution: Verify input values are within expected ranges
         Try the example test cases provided above
